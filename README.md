# Тестирование интернет-магазина upstore24.ru
Автоматизация тестирования основного пользовательского пути в интернет-магазине.

## О проекте
Автоматизированное тестирование ключевых сценариев работы интернет-магазина https://upstore24.ru/. 
Проверка полного цикла: от выбора товара через фильтры до оформления заказа в корзине.

## Тестируемые сценарии
- **Фильтрация товаров** в каталоге
- **Добавление товаров** в корзину  
- **Валидация данных** (название, цена)
- **Изменение количества** товаров в корзине
- **Оформление заказа**

## Технологии
- Python 3
- Selenium WebDriver
- Pytest
- Allure

## Установка зависимостей
pip install -r requirements.txt

## Запуск тестов
python -m pytest tests/ -v

## Запуск тестов с генерацией Allure отчетов
python -m pytest tests/ --alluredir=test_results/

## Просмотр отчета в браузере
allure serve test_results/
