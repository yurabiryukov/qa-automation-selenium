# Тестирование интернет-магазина upstore24.ru

Автоматизация тестирования основного пользовательского пути в интернет-магазине.

# О проекте
Автоматизированное тестирование ключевых сценариев работы интернет-магазина https://upstore24.ru/. 
Проверка полного цикла: от выбора товара через фильтры до оформления в корзине.

# Тестируемые сценарии
1. **Фильтрация товаров** в каталоге
2. **Добавление товаров** в корзину  
3. **Валидация данных** (название, цена)
4. **Изменение количества** товаров в корзине
5. **Оформление заказа**

# Технологический стек
- Python 3
- Selenium WebDriver
- Pytest

# Установка зависимостей
pip install -r requirements.txt

# Запуск тестов
python -m pytest tests/ -v
