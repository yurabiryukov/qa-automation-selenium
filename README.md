# Тестирование интернет-магазина

## Аннотация проекта
**Цель**: Проверка корректности основного бизнес-пути покупателя на примере сайта https://upstore24.ru/. От выбора товара через фильтры до добавления в корзину.

**Основные сценарии:**
- Фильтрация товаров в каталоге
- Добавление товаров в корзину
- Валидация данных (название, цена)
- Изменение количества товаров

# Установка зависимостей
pip install -r requirements.txt

# Запуск тестов
python -m pytest tests/ -v
